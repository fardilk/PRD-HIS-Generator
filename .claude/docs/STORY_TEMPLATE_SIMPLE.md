# Story Template - Simplified Version

**Simple story structure untuk generate PRD**
**Hanya 4 field utama di Story Overview**

---

## Format: stories/[MODULE_NAME]/story-[N].md

Struktur simplified untuk setiap story:

```markdown
## Story #[N]: [Story Title]

### Story Overview

**User Persona:**
[Siapa yang melakukan action ini]
=> Contoh: Pasien / Fisioterapis / Dokter

**Jobs to be Done:**
[Apa yang user ingin accomplish / yang ingin dicapai]
=> Contoh: Pasien bisa terdaftar ke sesi rehab medik dengan signature digital

**Referensi UI:**
[Figma link atau kosongkan dulu - akan diisi nanti]
=> Kosongin: [Akan diisi nanti]

**User Flow:**
[Detail akan diisi nanti - kosongkan dulu]
=> Kosongin: [Akan diisi nanti]

---

[Rest of detailed sections jika ada - tapi prioritas utama hanya 4 field di atas]
```
